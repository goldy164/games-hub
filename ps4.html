دليل تعديل الـ PS4 (من البداية للاحتراف)

كل ما تحتاج معرفته بخطوات بسيطة وبدون أي قلق أو توتر.

💡 متوتر وبتجرب لأول مرة؟

دي حاجة عادية جداً! كلنا كنا مكانك. المهم تتابع الخطوات بهدوء وتتأكد من رقم إصدار جهازك بدقة قبل ما تبدأ أي خطوة.

1. كيف تعرف إصدار جهازك؟ (الخطوة الأولى الإجبارية)

  • افتح جهاز الـ PS4 واכנס على الإعدادات (Settings).
  • انزل تحت خالص واختار System (النظام).
  • اختار System Information (معلومات النظام).
  • شوف رقم الإصدار المكتوب قدام System Software (مثلاً: 4.05، 9.00، إلخ).

2. البداية التاريخية: إصدار 4.05

ده يعتبر الحجر الأساس وأول إصدار استقر تماماً وشغل الألعاب بكل حرية بدون فلاشة USB نهائياً.

  • طريقة العمل: تعتمد على ثغرة متصفح ونواة صافية.
  • الخطوات: تفتح متصفح الجهاز (أو دليل المستخدم) وتدخل على صفحة الاستضافة (Host) الخاصة بالإصدار، وتنتظر ثواني حتى تظهر رسالة النجاح.

أشهر صفحات الاستضافات التاريخية لهذا الإصدار:

🔗 صفحة المصدر (SpecterDev) على GitHub

3. كيف تدخل وتنفذ الثغرة من متصفح الـ PS4؟

  • عطل التحديثات التلقائية فوراً من إعدادات الشبكة والنظام.
  • افتح متصفح الـ Internet Browser أو اذهب لدليل المستخدم (User's Guide) حسب الصفحة اللي بتدعم جهازك.
  • اكتب رابط صفحة الـ Host في المتصفح، أو استخدم صفحة محفوظة عندك.
  • اضغط على زر التفعيل (مثلاً زر الـ HEN أو GoldHEN) وانتظر حتى يكتمل التحميل ويظهر إشعار النجاح على الشاشة، ومبروك عليك الجهاز مهكر!
